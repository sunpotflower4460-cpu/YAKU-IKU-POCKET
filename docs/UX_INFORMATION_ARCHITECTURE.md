## タブ再構成（現4タブ → 意味の再定義）
1. **今日 (Today)** — 今日の観察 / 季節 / 最近の記録 / 観察チャレンジ / 学びカード
2. **観察 (Observe)** — 中央主要タブ。スキャン/写真追加/ライブカメラ/複数写真/観察ガイド
3. **探す (Explore)** — 図鑑/検索/地域/科・属/特徴検索/比較履歴
4. **記録 (Fieldbook)** — 観察記録/タイムライン/カレンダー/お気に入り/メモ/実績/設定

## 原則
タブは移動専用（実行ボタンにしない）/ 1画面1主役 / フィルターはBottom Sheet / 安全警告はAlert乱用せず画面内統合 / 戻るで入力が消えない。
