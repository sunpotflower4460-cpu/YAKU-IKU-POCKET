# プロダクトビジョン v2（PRODUCT_VISION_V2）

> 統合仕様書 §0, §2, §24 の要約。プロダクトの北極星。

## 一言
> 自然の中で植物を観察し、候補を比較し、見分け方・生態・季節・危険性を学び、自分だけのフィールドノートを育てる、ゲーム性のある植物観察アプリ。

## ポジション転換
| 旧 | 新 |
|---|---|
| AIが自動識別する | AI＋地域・季節・形態で複数候補を提示 |
| 食用/要注意/危険を色分類 | 危険回避の知識として安全情報を提供 |
| 新種を図鑑登録してXP | 観察を記録し「観察の旅」を育てる |
| 薬効を閲覧 | 伝統利用を出典付きで学ぶ |

## ブランドの核（5点）
日本の野草・ハーブへの親しみ / 続けやすいゲーム性 / 類似・毒草を学べる / 季節を感じる / 観察記録が美しく積み重なる。

## 禁止（§21 抜粋）
食用可否の断定 / 薬効の医学的断定 / 写真のみを根拠にした採取・摂取推奨 / ランダム結果 / 無検証のLLM出力使用 / 危険度を色だけで表現。

## コピー指針（置換表）
食用可→安全情報を確認、AI認識精度→候補一致度、図鑑に登録→観察記録として保存、発見→観察、効能→伝統的な利用記録、安全な収穫→安全観察チャレンジ。
