## 🔴 残タスク（開発者ご本人の対応が必須）

1. **アイコン / スプラッシュの差し替え**
   現状は空白のプレースホルダー（`assets/README.md` に明記、`icon.png`/`adaptive-icon.png` は約5KB）。Apple は空白アイコンをリジェクトします。仕様は `assets/README.md` 参照（icon 1024×1024 透過なし 等）。

2. **プライバシーポリシーの作成・ホスティング**
   App Store Connect は Privacy Policy URL が必須。作成後、App Store Connect に登録し、`src/constants/app.ts` の `PRIVACY_POLICY_URL` / `TERMS_URL` を実URLに差し替え。

3. **`eas.json` の提出用認証情報**
   `submit.production.ios` の `appleId` / `ascAppId` / `appleTeamId` が `YOUR_...` のまま。実値に置換しないと `eas submit` が失敗します。

4. **App Privacy「栄養ラベル」の設定**
   ※ 実AI導入後はカメラ画像を第三者API（Anthropic）へ送信するため、データ収集・第三者送信の申告が必要になります。

5. **年齢レーティング質問票**
   毒/死の記述・健康関連表現を正直に申告（4+ より上になる想定）。

6. **EAS `projectId` / `owner`**
   初回 `eas build` 実行時に `app.json` に付与されます。
