# assets/ — 画像仕様メモ

現在のファイルはすべて**プレースホルダー（無地）**です。
App Store 提出前に以下の仕様で差し替えてください。

| ファイル | サイズ | 形式 | 用途 |
|----------|--------|------|------|
| `icon.png` | **1024×1024px** | PNG（透過なし） | App Store アイコン |
| `splash.png` | **1284×2778px** | PNG | スプラッシュ画面（iPhone 14 Pro Max基準） |
| `adaptive-icon.png` | **1024×1024px** | PNG（透過あり） | Android アダプティブアイコン |
| `favicon.png` | **48×48px** | PNG | Web版ファビコン |

## デザインガイドライン
- 背景色: `#2E7D32`（メインカラー）または `#E8F5E9`（ライト）
- 中央モチーフ: 葉 / 野草 / ポケット図鑑をイメージするシンボル
- 作成ツール: Figma, Canva, Sketch など

## 注意
- icon.png は角丸・透過 NG（Apple が自動で角丸にする）
- adaptive-icon.png は中央 72% 以内にデザインを収める（セーフゾーン）
